get_avg_time ( $  stop_id,
$  route_id,

)

Queries the database and gets the average time it takes to get to a stop, assuming the bus is coming from the stop directly before this one.

Parameters:
int$stop_idID for the stop we want the average time of, if we want to know how long it takes for a bus to go from stop 2 to stop 3 on route 1, we would pass in 3 (average time to get to stop 3).
int$route_idStops can have multiple routes, use this to ensure we pull right time.
mysqli$mysqliAlready made mysqli object that is connected to dotCloud DB (making it is a mild pain).
Returns:
null|avg_time_to_next_stop Returns either an integer which represents the average time it will take for a bus, on the specified route, to get to the specified stop (in minutes) or returns NULL if query failed.
